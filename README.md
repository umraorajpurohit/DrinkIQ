# 🍺 DrinkIQ

**DrinkIQ** is a comprehensive web-based platform designed to promote responsible drinking through education, awareness, and practical tools. At its core, DrinkIQ helps users understand their alcohol consumption and its effects on their body, while building toward a supportive community for healthier lifestyle choices.

## 🎯 What Does DrinkIQ Do?

### Current Features

#### 1. **BAC (Blood Alcohol Concentration) Estimator**
- **Real-time BAC calculation** using the Widmark equation
- Input your drinks, body weight, sex, and drinking timeline
- Get estimated BAC levels and projected sobriety time
- Visual chart showing BAC elimination over time
- **Legal limit awareness** (India's 0.03% BAC limit clearly displayed)
- Clear warnings about driving safety

#### 2. **Comprehensive Drink Database**
Indian and international drinks categorised by type:
- **Beer** (Kingfisher, Bira, Corona, Heineken, etc.)
- **Wine** (Red, White, Sparkling)
- **Spirits** (Whisky, Vodka, Rum, Gin, Tequila)
- **Cocktails** (Old Fashioned, Mojito, Margarita, etc.)
- Accurate alcohol content (grams) for each drink

#### 3. **Hangover Recovery Guide**
Evidence-based information on:
- **What causes hangovers** (5 key mechanisms explained)
- **What actually helps** (hydration, electrolytes, nutrition, rest)
- **Common myths debunked** (coffee, cold showers, "hair of the dog")
- **Emergency recognition** (alcohol poisoning warning signs)
- **When to seek medical help** immediately

#### 4. **User Experience Features**
- **Dark/Light theme** toggle
- **History tracking** of past BAC calculations
- **User accounts** (optional, with guest mode available)
- **Mobile-responsive** design
- **Privacy-focused** (all data stored locally in browser)
- Clean, professional medical-grade UI

## 💡 Why Is DrinkIQ Helpful?

### For Social Drinkers

- **Reduce drunk driving** through clear time-to-sobriety estimates

- **Combat misinformation** with evidence-based content
- **Promote moderation** through awareness and tracking

## 🛠️ Technical Stack

### Current
- **Frontend**: Vanilla HTML, CSS, JavaScript
- **Storage**: LocalStorage (client-side)
- **Hosting**: Vercel
- **Design**: Custom CSS with responsive design

### Planned
- **Backend**: Node.js + Express (for user accounts, community)
- **Database**: PostgreSQL (user data, blog posts) + Redis (session management)
- **Authentication**: OAuth 2.0 (Google, phone number)
- **APIs**: 
  - Uber, Ola, Rapido APIs for cab integration
  - Google Maps API for geolocation
  - Twilio for SMS notifications
- **CMS**: Strapi or custom solution for blog management
- **AI/ML**: OpenAI API for chatbot, TensorFlow for pattern analysis
- **Payment**: Razorpay (for premium features, donations)

## ⚠️ Important Disclaimers

- DrinkIQ is **NOT** a medical device or breathalyzer
- BAC estimates are **approximate only** and should not be used for driving decisions
- Individual metabolism varies significantly
- **Always err on the side of caution** - when in doubt, don't drive
- This tool is for **informational and educational purposes only**
- Seek professional medical help for alcohol dependency
- In emergencies, always call local emergency services (112 in India)
